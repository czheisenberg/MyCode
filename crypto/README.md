## HOW  TO USE?

- ImgBase64.py

- 图片-base64编码互转。
- linux/windows 均可运行。

```shell
python3 ImgBase64.py --help
```

![1](imgs/1.png)

图片转base64

```
python3 ImgBase64.py -i ./imgs/gakki.jpg
```

![2](imgs/2.png)

base64转图片

```
python3 ImgBase64.py -b ./output/base64/gakki.txt
```
